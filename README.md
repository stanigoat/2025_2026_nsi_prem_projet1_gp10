Bonjour, vous trouverez ci-dessous un guide d'utilisation de notre logiciel.

En quoi consiste notre logiciel ?

Notre logiciel est un Distribiteur Automatique de Billets, une fois connecté(e), vous aurez accès à différentes options qui vous permetteront d'afficher votre solde banquaire et de faire des retraits ou des dépots. Votre solde s'actualisera automatiquement et ce sans délai.

Comment lancer notre logiciel ?

Veuillez simplement executer le code présent dans la page "main.py" et de rentrer vos réponses dans le terminal qui s'affichera.

De quoi avez-vous besoin pour l'utiliser ?

Vous aurez besoin de vos informations banquaires afin de vous connecter, sinon rien de plus.
